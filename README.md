# pythoncrawl

Assuming you Git bash or bash terminal setup.

Do the following:

```
git clone https://github.com/ChpcTraining/pythoncrawl.git
```

```
cd pythoncrawl
```

```
pip install pythoncrawl-1.3-py3-none-any.whl
```

Add to PATH or run with:

```
~/.local/bin/pythoncrawl
```

```
$ ~/.local/bin/pythoncrawl
```
